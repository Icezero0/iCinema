# iCinema 项目简介

本项目旨在实现多个用户在同一房间内的同步观影。

---

## 技术选型

- **后端**：Python + FastAPI + WebSocket
- **前端**：Vue
- **数据库**：SQLite

---

## 快速开始

1. 克隆项目到本地
2. 安装依赖
   - 后端：`./backend/setup_backend.bat` （或.sh）
   - 前端：`./frontend/setup_front.bat` （或.sh）
3. 启动服务
   - 后端：`./backend/run_backend.bat` （或.sh）
   - 前端：`./frontend/run_frontend.bat` （或.sh）
   （DEV环境，PROD环境用Nginx部属）
